## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

-   node
-   npm

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/Carlos-E-Souza/To-Do-List.git
    ```
2. Open the folder
    ```sh
    cd To-Do-List
    ```
3. Install NPM packages
    ```sh
    npm install
    ```
4. Then run the local server with
    ```sh
    npm run dev
    ```
